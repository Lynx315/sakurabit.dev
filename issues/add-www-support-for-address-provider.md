### Issue Title: Add WWW Support for Address Provider

#### Description:
This issue is to implement support for the 'www' prefix in the Address Provider service, allowing users to access the service through both 'www.example.com' and 'example.com'.

#### Acceptance Criteria:
- The service should recognize requests coming from both domains.
- Ensure that any necessary redirects are in place.
- Test that both versions of the URL return the correct results.

#### Additional Notes:
This enhancement will improve accessibility and user experience.
